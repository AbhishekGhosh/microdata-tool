Microdata Tool

A drop-in script for browsing and debugging HTML5 microdata.

So, you're implementing HTML5 microdata to improve your site's findability, but find it annoying to use Google's Webmaster Tools to verify your results, having to republish your site for each and every change?

Been there. That's why I've built a little jQuery drop-in script that you can include on your page and have a handy browser/validator handy, even without Internet connectivity or even a web server.

See that little white box in the left hand corner? Take a look.

Want to start using it? Go grab the code!

Wanna try it out? Hover over each of the items in the white box to highlight the relevant HTML elements in the document and click to see each object's properties.
